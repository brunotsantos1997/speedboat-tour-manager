# 🚤 Sistema de Gestão de Passeios

**Sistema de gestão de passeios de lancha com vouchers e reservas online.**

## 📋 **Visão Geral**

Sistema web completo para gestão de passeios turísticos de lancha, desenvolvido com React 19, TypeScript e Firebase. Oferece controle total de eventos, clientes, embarcações, finanças e comissões com autenticação segura e interface responsiva.

### 🎯 **Funcionalidades Principais**

- **📅 Gestão de Eventos**: Criação e gerenciamento de passeios com calendário integrado
- **👥 Gestão de Clientes**: Histórico completo e cadastro durante reservas
- **⛵ Gestão de Embarcações**: Controle da frota com informações detalhadas
- **💰 Sistema Financeiro**: Controle de despesas, livro caixa e relatórios
- **🎫 Sistema de Vouchers**: Geração de vouchers personalizados para eventos
- **👥 Gestão de Usuários**: Sistema de permissões com aprovação administrativa
- **💳 Comissões**: Cálculo automático e relatórios de comissões
- **📊 Dashboard**: Métricas e visualizações em tempo real
- **🔐 Autenticação Segura**: Login, signup e recuperação de senha
- **📱 PWA**: Aplicativo instalável com suporte offline
- **📱 Interface Responsiva**: Otimizada para dispositivos móveis
- **🌍 Internacionalização**: Suporte a múltiplos idiomas (PT-BR, EN-US, ES-ES)

## 🛠️ **Stack Tecnológico**

### **Frontend**
- **React 19** + **TypeScript** - Componentes modernos e type-safe
- **Vite** - Build ultra-rápido com Hot Module Replacement
- **Tailwind CSS** - Design system responsivo
- **React Router DOM** - Navegação client-side com lazy loading
- **Lucide React** - Biblioteca de ícones

### **Backend & Database**
- **Firebase** - Autenticação, Firestore e Storage
- **Firebase Authentication** - Sistema de login seguro
- **Firestore** - Database NoSQL com regras de segurança

### **Bibliotecas Principais**
- **TipTap** - Editor de texto rico para descrições
- **React Day Picker** - Calendário interativo
- **DOMPurify** - Sanitização de HTML contra XSS
- **html2pdf.js** - Geração de PDFs de vouchers
- **React Input Mask** - Máscaras para formulários
- **React Joyride** - Tours interativos guiados
- **UUID** - Geração de IDs únicos
- **bcryptjs** - Hash de senhas
- **date-fns** - Manipulação de datas
- **i18next** - Sistema de internacionalização
- **react-i18next** - Integração de i18n com React

### **Desenvolvimento & Testes**
- **ESLint** - Linting e qualidade de código
- **Playwright** - Testes end-to-end automatizados
- **Vitest** - Testes unitários
- **TypeScript** - Tipagem estática rigorosa

## 🚀 **Início Rápido**

### **Pré-requisitos**
- Node.js 18+
- npm
- Conta Firebase configurada com Firestore

### **Instalação**
```bash
# Clonar repositório
git clone <repository-url>
cd Voucher-passeio-lancha

# Instalar dependências
npm install

# Configurar ambiente
cp .env.example .env
# Editar .env com suas credenciais Firebase

# Iniciar desenvolvimento
npm run dev
```

### **Variáveis de Ambiente**
```env
# Firebase
VITE_FIREBASE_API_KEY=sua_api_key
VITE_FIREBASE_AUTH_DOMAIN=seu_auth_domain

VITE_FIREBASE_MESSAGING_SENDER_ID=seu_messaging_sender_id
VITE_FIREBASE_APP_ID=seu_app_id
VITE_FIREBASE_MEASUREMENT_ID=seu_measurement_id

```

## 🌍 **Internacionalização (i18n)**

### **Idiomas Suportados**
- 🇧🇷 **Português (Brasil)** - `pt-BR` (idioma padrão)
- 🇺🇸 **English (United States)** - `en-US`
- 🇪🇸 **Español (España)** - `es-ES`

### **Funcionalidades**
- **Detecção Automática**: Baseada no navegador do usuário
- **Persistência**: Escolha salva em localStorage
- **Formatação Localizada**: Datas, moedas e números
- **Componente de Seleção**: Interface para mudança de idioma
- **Hook Personalizado**: `useLanguage` para formatação avançada

### **Uso Básico**
```typescript
import { useLanguage } from './ui/hooks/useLanguage';

function MyComponent() {
  const { t, formatCurrency, formatDate } = useLanguage();
  
  return (
    <div>
      <h1>{t('dashboard.title')}</h1>
      <span>{formatCurrency(1500.50)}</span>
      <span>{formatDate(new Date())}</span>
    </div>
  );
}
```

### **Adicionando Novos Idiomas**
1. Criar arquivo em `src/i18n/locales/`
2. Configurar em `src/i18n/index.ts`
3. Atualizar `LanguageSelector.tsx`
4. Adicionar traduções em todos os módulos

## 🔐 **Segurança**

### **Implementações**
- **Firebase Authentication**: Sistema de autenticação seguro
- **Firestore Security Rules**: Regras detalhadas de acesso por coleção
- **Role-Based Access Control**: Múltiplos níveis de permissão
- **Input Sanitization**: Proteção contra XSS com DOMPurify
- **Password Hashing**: Senhas criptografadas com bcryptjs
- **Protected Routes**: Rotas protegidas por nível de acesso

### **Níveis de Acesso**
- **OWNER**: Acesso completo ao sistema
- **SUPER_ADMIN**: Gestão administrativa completa
- **ADMIN**: Gestão de usuários e operações
- **SELLER**: Acesso a vendas e dashboard

### **Regras de Segurança Firestore**
- Validação de autenticação em todas as operações
- Verificação de status de aprovação do usuário
- Controle granular por coleção e documento
- Funções auxiliares para verificação de permissões
